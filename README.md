# FBLA-Project
2026 FBLA Entry. For Schooling Purposes Only.

This code is made entirely by Amari Gibson and for Academic Purposes (FBLA Competition).

The process goes as follows:

1. User signs up for program
  a. Username & password provided on page
  b. two-step verification with email asked and provided
2. User chooses a method to sort businesses
  a. User sorts by business name (A-Z)
  b. User sorts by business type (food, retail)
  c. User sorts by business by ratings
3. Business page for user to view or favorite
  a. contains business name
  b. contains business rating
  c. contains business description & deals
  d. contains section to leave a rating/comment
4. Settings page for accessibility feature
